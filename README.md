# www.portalmc.org

The webpage repository of www.portalmc.org.

## Development

### Watch changes and run server

```
./gradlew build -t
# Ctrl-D to stop

# in other terminal
cd public
python3 -m http.server
```

### Build pages for gh-pages

```
./gradlew build
```
